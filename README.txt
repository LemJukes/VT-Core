README
Overview
This project contains a JavaScript application that displays the current time in various formats, including a human-readable format using words. The time updates every second.

Files
index.html: The HTML file that includes the script and displays the time.
script.js: The main JavaScript file that updates the time every second.
verbatempus-core.js: A JavaScript module that provides the function to convert the current time into words.
Installation
Clone the repository or download the files to your local machine.

Ensure you have a web server to serve the files. You can use a simple HTTP server like http-server for Node.js.

Usage
Start the HTTP server in the directory containing the files.

Open your web browser and navigate to http://localhost:8080 (or the port specified by your HTTP server).

The current time will be displayed in various formats, including the human-readable format using words.

File Descriptions
index.html
This file includes the necessary HTML structure and references the script.js file.

script.js
This file contains the main logic to update the time every second and display it in different formats.

verbatempus-core.js
This file contains the logic to convert the current time into words.
